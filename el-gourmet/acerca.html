Práctica 1 de la asignatura Programación del Cliente Web de 2º curso de Ingeniería Multimedia. Esta práctica trata de la realización de un sitio web de recetas de comida en la que los usuarios pueden registrarse para subir sus recetas e interactuar con las de otros usuarios. El objetivo de esta práctica es aprender los fundamientos del lenguaje HTML y CSS para el desarrollo de páginas web.

Implementación:
  • Las páginas tienen todas una parte común, que es la cabecera, barra de navegación y el pie de página.
  • Respecto al CSS, el campo que recibe el foco destaca, pero no se han utilizado pseudo-elementos ni pseudo-clases.
  • Se han utilizado las Web Fonts de Google, tres tipos de fuentes, que se pueden observar por ejemplo en el título de la web y su descripción.
  • Los Icon Fonts de fontello se han utilizado para los elementos de la barra de navegación.
  • Se ha utilizado la técnica Mobile First para realizar un diseño responsive como dice el enunciado de la práctica. No incluye menú hamburguesa para la barra de navegación
  • Respecto a la información de cada página, se han realizado todos los puntos que especifica la práctica excepto en la página receta.html, donde no se ha implementado los dos últimos puntos, el mostrar todas las recetas del autor al clicar sobre el nombre del autor, la zona de comentarios y los botones de me gusta/no me gusta.
